# emacs_config

* ac-emmet
* editorconfig
* evil
* flycheck
* git-gutter
* js2-mode
* js2-refactor
* lua-mode
* markdown-mode
* material-theme
* neotree
* php-auto-yasnippets
* php-mode
* shell-pop
* web-mode
* yasnippet

## Installation

* `git clone https://github.com/bcbcarl/emacs_config.git ~/.emacs.d`
* `tic -o ~/.terminfo /usr/share/emacs/22.1/etc/e/eterm-color.ti`
